# 實作記錄 (Walkthrough) - 導航熱區修正與 T 字提示線 (第 8 版)

本次開發將導航提示 UI 進一步精煉，透過「T 字型」設計在提供明確指引的同時，最大程度地減少了對樂譜內容的干擾。

## 變更項目

### 1. T 字型發光提示線 (`InputManager.js`, `interaction.css`)
- **設計理念**：不使用貫穿全螢幕的線條，改為在比例交界點繪製 T 字圖形。
- **座標與比例**：
  - **交會點**：位於 PDF 頁面高度 35% 與 寬度 40% 的相交處。
  - **尺寸限制**：橫向與縱向線條的長度均限制為該軸向總長度的 **1/5 (20%)**。
- **視覺效果**：採用漸層發光 (`linear-gradient`)，線條向兩端逐漸透明，減少突兀感。

### 2. 動態追蹤與對齊
- **即時更新**：T 字的位置會隨 PDF 縮放、側邊欄開啟/關閉、手勢縮放等行為動態重新計算，確保始終精確指向熱區邊界。
- **阻斷拖曳**：維持 10 像素位移檢查，確保拖曳操作不誤觸熱區。

### 3. 三向方向指示 (延續優化)
- **頂部**：向上箭頭。
- **左下**：向左箭頭。
- **右下**：向右箭頭。
