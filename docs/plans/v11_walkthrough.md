# 實作記錄 (Walkthrough) - 導航熱區與動態 T 字提示 (第 11 版)

本次開發針對觸發回饋的節奏進行了微調，使 UI 響應感覺更加靈敏且輕量。

## 變更項目

### 1. 閃爍時間優化 (`InputManager.js`)
- **時間調整**：將 T 字型提示線在觸發換頁時的顯示時間從 1 秒縮短為 **0.5 秒 (500ms)**。
- **優點**：更短的時間能讓回饋感顯得更「清脆」，且能更快讓視覺中心回到樂譜內容上，減少視覺殘留。

### 2. 精煉功能複測 (延續第 10 版)
- **真正的 T 字型**：縱向線條僅從橫向交點向下延伸，保持標準 T 型。
- **動態追蹤**：T 字位置在 PDF 縮放或側邊欄切換時依然能精確維持在熱區界線上。
- **三向指示器**：正確的上/左/右箭頭動畫回饋。
- **拖曳阻斷**：滑鼠拖曳操作（位移 > 10px）不會誤觸 0.5 秒的閃爍。
