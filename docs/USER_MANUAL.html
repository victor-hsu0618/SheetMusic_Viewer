1. 快速開始

開啟樂譜

  1. 點擊首頁的 Open Solo Score，選取 PDF 檔案。
  2. 樂譜載入後，工具列（右側浮動欄）與尺規（左側）自動出現。
  3. 首次開啟時，系統在背景自動偵測五線譜行位置。

3. Jump 導航系統

Jump 是 ScoreFlow 最核心的導航功能，設計給演奏中或練習時快速翻頁。

3.1 跳躍線 (Jump Line)

視窗中有一條水平虛線，定義了視線焦點的位置。Jump 後，目標內容會對齊在這條線上。

3.2 Jump 前進邏輯

① Anchor 存在 → 跳至跳躍線下方最近的 Anchor
② System 存在 → 跳至「頁面重疊設定」指定的 System
③ 最終 fallback → 捲動 (viewport 高度 − 跳躍線偏移量)

10. PWA 安裝與離線使用

ScoreFlow 是一款 PWA (Progressive Web App)，它可以像原生應用程式一樣安裝到您的桌面上或手機主畫面上，並且在完全沒有網路（例如演藝廳、飛機上）的情況下仍能正常運作。

10.1 為什麼要安裝成 App？

  • 全螢幕體驗：移除瀏覽器網址列，將所有視窗空間留給樂譜。
  • 離線啟動：即使沒有 Wi-Fi，也能從主畫面直接開啟 App 並讀取樂譜。
  • 快取優先：核心引擎會被永久快取在您的設備中，提升啟動速度。

10.2 安裝指南

iOS / iPadOS (Safari)

  1. 在 Safari 中開啟 ScoreFlow 網址。
  2. 點擊工具列底部的 分享 (Share) 圖示（箭頭向上方框）。
  3. 選擇 加入主畫面 (Add to Home Screen)。

Android / Chrome / Windows / macOS

  1. 開啟 ScoreFlow 後，點擊網頁列右側的 安裝 字樣。
  2. 或者點擊瀏覽器右上角的 三個點 (⋮) 並選擇 安裝 ScoreFlow。

Edge (Windows/macOS)

  1. 點擊網址列右端的 應用程式可用圖示 進行安裝。

10.3 離線使用注意事項

首次快取：建議在有網路時至少開啟一次 ScoreFlow 並載入您的常用樂譜，確保資源已妥善快取。

ScoreFlow 將樂譜儲存在瀏覽器的 IndexedDB 中。請勿清除瀏覽器的快取資料，否則本機樂譜需重新手動開啟。
