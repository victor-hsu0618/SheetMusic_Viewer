🧹 Score Cleaner

清除影印髒污，重建五線譜與音符

清理模式

數值越低，越多灰色被判為黑色（線條更多）；越高，只保留最深的黑色。

拉大黑白之間的差距。灰色背景嚴重時先調高對比，再套用二值化效果更好。

用模糊平滑消除細小雜點。值越高越模糊，可能使細線或小音符變淡，建議用 1–2。

影響處理速度與輸出精細度。300 dpi 適合列印，200 dpi 適合螢幕閱讀。

五線譜加強

掃描每行的深色像素比例，找出橫跨整頁的水平線條（五線譜），並將其塗成純黑色，修復因影印造成的斷線或淡色。

綠色＝將加強，紅色＝已排除。點擊線條切換。青色終止線可左右拖曳手動調整。

該行有多少 % 的像素是深色才算五線譜。太低會把文字也誤抓；太高可能漏掉淡色的線。

0 = 自動沿用偵測到的粗細；1–10 = 強制指定像素寬度，以線段中心為基準重畫。

進階 System 偵測設定

每行暗像素比率超過此值才視為高密度（五線譜）行。

在頁面左側多少 % 範圍內搜尋連結兩行的垂直 Bracket 線。

超過此距離的群組絕對不合併（防止跨 System 誤合）。

音符頭清理

找出符合橢圓形狀的黑色連通區域，判斷是實心（四分音符）或空心（二分音符），重新畫成乾淨的幾何橢圓形。

太小會把雜點誤認為音符。若漏抓小音符，往左調低。

太大會把文字區塊或休止符誤判。依樂譜字體大小調整。
