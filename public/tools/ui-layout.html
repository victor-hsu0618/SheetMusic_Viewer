💬 Communication Vocabulary

使用以下專有名詞可以讓我們溝通更精準：

Term (名詞) Description (描述)
Doc Bar (Document Bar) 底部那個可以拖動、收合的黑色工具欄。包含頁碼控制與主要面板開關。
Floating Stamp Bar 選取圖章工具後，畫面中央出現的半透明圓角工具欄（含有各類表情符號/標記）。
Jump Ruler 畫面右側的垂直尺規，顯示下一個 Jump 的基準點與歷史紀錄。
Manager 後台負責控制 UI 邏輯的 JavaScript 類別（如 DocBarManager）。
Inspector / Detail Panel 側邊或漂浮出現的詳細設定面板（如 Score Detail 或 View Inspector）。
