
**PWA 樂譜瀏覽標記軟體 - 工具列設計完整文件（重新整理版）

以下是完整工具列設計整合方案，涵蓋 PWA 架構、iOS/PC 快捷鍵盤、群組功能、手機遠端等全部需求。

一、 專案概述與整體架構

本方案採用「三層Robbin響應式工具列」設計，確保在不同裝置上都能提供一致且高效的操作體驗。

請參考以下新的排版建議：

1.1 三層響應式工具列佈局

本方案將工具列劃分為三個層次，以確保在不同裝置上都能提供一致且高效的操作體驗：

* 第一層 Stamp 主導航 ( 按 Stamp 工具後顯示)
  * 此層級為主要功能入口，doc controlcenter左側顯示。
  * 內容範例：
    1. Stamp 主群組 1️⃣Edit | 2️⃣Pens | 3️⃣Bowing | …
    2. 數字鍵盤 (固定)
       1. 提供快速選取工具群組及子工具的功能。
       2. 觸發方式: 長按主導航列上的 🔢 圖示。
       3. 佈局 (3x3 按鍵)：
  * 功能: 主要模式切換與工具群組入口。
* 第二層 Stamp工具列 (點擊展開)
* 此層級為實際的標記工具，需點擊主導航列中的工具群組後展開。
* 內容範例： 依選定群組顯示內容
* 功能: 核心標記與操作工具。

二、 核心功能：工具群組對應表

所有標記功能被劃分為 8 個群組，並各自配備數字鍵、PC 字母鍵三種快捷操作。

| 群組         | 數字 | PC字母 | 子工具範例             |
| ------------ | ---- | ------ | ---------------------- |
| Edit         | 1    | E      | 刪除/選取              |
| Pens         | 2    | P      | 黑/紅/藍筆/橡皮擦/高亮 |
| Bowing       | 3    | B      | 上弓/下弓/弓法符號     |
| Fingering    | 4    | F      | 拇指 , 1-5指法...      |
| Articulation | 5    | A      | Staccato/Legato/Accent |
| Tempo        | 6    | T      | ♩=60/♩=80            |
| Dynamic      | 7    | D      | pp/p/mp/f/ff           |
| Layout       | 8    | L      | Anchor/小節            |

三、 使用者操作與控制

提供針對 iOS、PC 和手機遠端三種環境的最佳化操作流程。

3.1 iOS 階層操作（iPad/iPhone）

1. 按Stamp工具列 → 數字鍵盤滑入（3x3大按鍵）。
2. 按「3」 → Bowing面板展開。
3. 按「1」 → 選取「上弓」工具。
4. Stamp 浮現於指標左上角（偏移 20px 以防手指遮擋）。
5. 觸控拖拽微調 → 離開觸控自動定位。

3.2 PC 雙快捷模式

提供多層次的快捷鍵操作：

* 單按字母：E → Edit 面板展開。
* 按住字母：B → 直接顯示 Bowing 群組列表。
* 定位操作：Mouse/方向鍵微調 。

3.4 手機遠端控制

允許手機作為樂譜瀏覽時的遠端控制器：

* 連接：掃描 QR Code → WebSocket 同步。
* 大按鈕區：[⬅️翻頁] [➡️翻頁] [🔢數字鍵盤] [▶️播放]。
* 數字鍵盤：功能同 iPad（1=Edit, 2=Pens, 3=Bowing...），並有觸控拖拽輔助。
* 音量鍵：音量減 (↓) = Undo / 音量增 (↑) = Redo。

四、 Stamp 浮動與定位機制

確保 Stamp 符號在放置時的精準度與操作順暢度。

| 裝置   | 浮動狀態   | 定位操作                            |
| ------ | ---------- | ----------------------------------- |
| PC     | 指標左上角 | 方向鍵/滑鼠微調 → Enter 確定定位   |
| iPad   | 指標左上角 | 觸控拖拽微調 → 離開觸控自動定位    |
| iPhone | 指標左上角 | 雙指縮放 + 拖拽 → 單指點擊確定定位 |

五、 視覺與操作回饋

提供使用者明確的視覺和觸覺回饋，優化操作體驗。

* 按鈕顯示：同時顯示數字和字母，如 "3/B"。
* Hover提示：提供詳細提示資訊，如 "Bowing: 3 / B  → ↑↓弓法"。
* 選定狀態：浮動 Stamp 時提供放大鏡預覽。
* 定位成功：短暫綠色閃光 + 震動回饋（iOS 裝置）。
