## 1. 快速開始

### 安裝為 PWA（推薦）
1. 在 iPad 上以 **Safari** 開啟 ScoreFlow 網址。
2. 點擊分享按鈕 → 「**加入主畫面**」。
3. 從主畫面圖示開啟，即可享有全螢幕與離線體驗。

### 第一次開啟
- 畫面會顯示歡迎畫面與教學手冊。
- 點擊右上角的「**📂 載入樂譜**」按鈕，或直接從側邊欄的樂譜庫選擇。

---

## 2. 載入樂譜

### 方式一：直接上傳（iPad / Mac）
1. 點擊工具列上的「**📂**」按鈕。
2. 選取本機 PDF 檔案。
3. 系統自動計算 SHA-256 指紋，確保此份樂譜的所有標記獨立儲存。

### 方式二：從樂譜庫開啟
1. 點擊左側邊欄的「**📚 Library**」圖示（或按 `O` 鍵）。
2. 點擊任何已儲存的樂譜卡片即可載入。

### 自動恢復
- 每次開啟 ScoreFlow，系統會自動還原**上次開啟的樂譜**及其捲動位置。

## 3. 瀏覽樂譜

### iPad 捲動
| 操作 | 效果 |
|---|---|
| **單指上下滑動** | 在 View 模式下：觸發 Zone Tap 跳轉（見下方）。在標記模式下：一般捲動。 |
| **雙指上下滑動** | 任何模式下都可以自由捲動 |
| **雙指左右快速滑動 (Swipe)** | 跨頁跳轉（向左 = 往後翻、向右 = 往前翻）|

### Zone Tap（智慧點擊跳轉）— 僅 View 模式
在**瀏覽模式 (View)**下，點擊特定區域會觸發跳轉：

```
┌──────────────────────────────┐
│  ← 左側 40%   │  右側 →      │
│   點擊 = 回跳  │  點擊 = 前跳 │
│                │              │
│  （上方 35%）  │              │
└──────────────────────────────┘
```

- **左上 35%**：回到上一個 Anchor / 小節
- **右側 60%**：跳至下一個 Anchor / 小節

## 4. 縮放與檢視

### iPad 雙指縮放
| 操作 | 效果 |
|---|---|
| **雙指張開／夾攏** | 放大 / 縮小樂譜 |
| **雙指平移（先移動再開合）** | 捲動頁面（不觸發縮放）|
| **雙指開合（先開合再移動）** | 縮放（不觸發捲動）|

> **提示**：系統採用「**手勢鎖定 (Gesture Locking)**」機制—雙指落下後，優先偵測到「移動」還是「開合」，即鎖定為「捲動」或「縮放」模式，防止互相誤觸。

### 快速縮放
- **適合寬度 (Fit Width)**：工具列的「⟺」按鈕，將樂譜縮放至符合螢幕寬度。
- **Mac**：`Cmd +` / `Cmd -`

## 5. 標記工具 (Stamps)

### 選擇工具
1. 點擊右下角的「**編輯工具列**」圖示展開工具箱。
2. 選擇要放置的標記符號（如 ↑ 上弓、♯ 升號、1–5 指法等）。

### 放置標記
1. 選好工具後，**按住螢幕不放**，一個半透明的預覽圖示（Ghost）會出現。
2. **移動手指**至目標位置，預覽圖示會即時跟隨。
3. **放開手指**，標記永久落點。

### Grace Period 微調（放下後仍能移動）
放下標記後，您有 **1.8 秒**的猶豫期（Grace Period）來調整位置，並有**兩種方式**可以微調：

#### 方式 A：靠近拖曳
- 在 1.8 秒內，用手指**點在標記附近**，就會直接進入拖曳模式。
- 畫面上會出現**垃圾桶** 🗑️，將標記拖入即可刪除。

#### 方式 B：Nudge（螢幕任意位置微移）★ 推薦
- 在 1.8 秒內，在**螢幕的任意位置**按住並移動。
- 手指的相對偏移量會直接轉換為標記的座標偏移。
- **適合場景**：放下標記後，大拇指在螢幕角落微幅拖動即可精準微調，完全不遮掩樂譜。

### 編輯工具
| 工具 | 功能 |
|---|---|
| **Select (選取)** | 點選已存在的標記進行拖動搬移 |
| **Copy (複製)** | 點擊已存在的標記，複製一份並進入拖曳模式 |
| **Cycle (輪轉)** | 點擊標記，在同組群組內切換類型（如 ↑/↓ 弓法、1-5 指法） |
| **Eraser (橡皮擦)** | 塗抹或點選已存在的標記進行刪除 |

### 智慧垃圾桶
- 拖動任何標記時，螢幕上方會自動出現垃圾桶。
- 拖至垃圾桶範圍內並放開 → **立刻刪除**。

## 6. 繪筆工具 (Pens)

- 從工具箱切換至「**Pens**」類別。
- 可選擇：**鋼筆、粗筆、螢光筆、毛筆、鉛筆、修正筆**等多種筆刷。
- **Apple Pencil**：識別筆壓，支援傾斜角度。

> **注意**：繪筆工具放置後**不進入 Grace Period**，以防止書寫時的誤觸修改。

## 7. 圖層管理 (Layers)

ScoreFlow 採用**五大專業圖層**架構，各圖層的標記**完全獨立**：

| 圖層名稱 | 預設用途 |
|---|---|
| **Performance** | 演奏詮釋記號（力度、速度提示）|
| **Fingering** | 指法（數字 1–5）|
| **Bowing** | 弓法（上弓 ↑、下弓 ↓）|
| **Text** | 文字標注、便利貼 |
| **Others** | 形狀、自由記號、Anchor |

### 切換圖層
- 點擊工具列上的圖層標籤（Layer Tabs），或從設定面板選擇。
- 切換圖層後，工具箱會**自動切換**至該圖層對應的符號組。

### 隱藏圖層
- 在圖層管理介面，點擊圖層右側的「眼睛」圖示即可隱藏 / 顯示。
- 隱藏的圖層在樂譜上完全不可見，適合「只看弓法」或「演出模式」使用。

## 8. 智慧跳轉系統

### Anchor 錨點
- 切換至 **Anchor** 工具，在樂句起點點擊放置。
- 畫面左側會在對應位置顯示小三角形 ▷ 標記。

### Jump 跳轉
- 按下**空白鍵**（Mac）或點擊工具列的**跳轉按鈕**，立即跳到**下一個 Anchor**。
- 跳轉對齊的基準是畫面上可見的**水平虛線（Jump Line）**。

### 調整 Jump Line 高度
- 在設定面板中調整「**Jump Offset**」，即可移動虛線至畫面任意高度。
- 建議對齊至「指揮棒所在的視線高度」。

## 9. 書籤與頁面跳轉

### 開啟跳轉面板
- 點擊工具列中的「**頁碼按鈕**」，或按鍵盤 `G` 鍵。

### 數字鍵盤跳轉
1. 在彈出的計算機面板輸入頁碼。
2. 按「**GO**」立刻跳至該頁。

### 書籤
1. 捲到目標頁面。
2. 開啟跳轉面板 → 切換至「**Bookmarks**」分頁 → 點擊「+ 新增書籤」。
3. 輸入名稱並儲存。
4. 下次點擊書籤列表，即可一鍵跳回。

### 小節跳轉（Measures Tab）
- 若您已在樂譜上放置了**小節號碼**標記，可在「**Measures**」分頁直接點擊小節號碼跳轉。

## 10. 斗篷標籤 (Cloak)

斗篷標籤讓您可以在**不同演奏場合**切換需要顯示的標記集合。

- 每個標記可被標記為「**黑 / 紅 / 藍**斗篷之一」。
- 在工具列切換斗篷可見狀態，例如：
  - **排練模式**：顯示全部（黑 + 紅 + 藍）。
  - **正式演出**：只顯示黑色（最必要的提示）。

## 11. 匯入與匯出

### 匯出 PDF（帶標記）
1. 點擊「**⋯ More**」→「**Export PDF**」。
2. 系統將原始樂譜與所有可見圖層的標記壓平合併為一份 PDF。

### 匯出 JSON（備份）
- 點擊「**Export JSON**」，將所有標記、圖層設定匯出為純文字 JSON 檔案。
- 可傳送給他人作為協作基底，或自行備份。

### 匯入 JSON（還原備份）
- 點擊「**Import JSON**」，選取備份的 JSON 檔案，標記會**疊加**到當前樂譜。

## 12. 鍵盤快捷鍵（Mac）

| 按鍵 | 功能 |
|---|---|
| `Space` / `J` / `↓` | 跳至下一個 Anchor（前跳）|
| `K` / `Shift+Space` / `↑` | 跳至上一個 Anchor（回跳）|
| `G` | 開啟／關閉跳轉面板 |
| `O` | 開啟／關閉樂譜庫 |
| `E` | 展開／收合編輯工具列 |
| `S` | 開啟／關閉設定面板 |
| `V` | 開啟／關閉檢視面板 |
| `R` | 開啟／關閉垂直尺規 |
| `F` | 全螢幕切換 |
| `D` | 顯示／隱藏 Doc Bar |
| `Cmd +` / `Cmd -` | 放大 / 縮小 |
| `Cmd Z` | 復原 (Undo) |
| `Cmd Shift Z` | 重做 (Redo) |
| `Esc` | 關閉最上層的面板 |
